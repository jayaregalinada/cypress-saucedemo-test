# Cypress Activity

## Requirements
- __Node JS__ ~12

## Installation

Run using [npx](https://www.npmjs.com/package/npx)

```sh
npx cypress open
```

For more ways to install cypress you can go to [https://cypress.io](https://docs.cypress.io/guides/getting-started/installing-cypress)
